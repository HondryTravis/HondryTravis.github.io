写博客的原因​

方便自己记忆，以前各大平台也都做过，维护困难，另外是对自己知识点的总结，很多用过长时间不用就忘记了，当个知识体系

  1. 不准说不行,尽可能去实现
  2. 要习惯性迭代自己的知识体系和写过的代码
  3. 不要停止学习

只要开始，虽晚不迟。尽人事，听天命
